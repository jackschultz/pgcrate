PostgreSQL companion CLI

pgcrate

Migrations, introspection, diffing, health checks, snapshots, and safe anonymized datasets — in a single binary.

Install GitHub crates.io
cargo install pgcrate

Core

  • Pure SQL migrations (up/down)
  • Schema introspection and diff
  • Health checks with JSON output

Data Ops

  • Snapshot save/restore (custom/plain)
  • Profile-based snapshot filtering
  • Anonymize dump + bootstrap workflows
